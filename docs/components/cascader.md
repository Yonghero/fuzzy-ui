# Cascader 级联选择器

当一个数据集合有清晰的层级结构时，可通过级联选择器逐级查看并选择

## 基本使用

:::demo
cascader/basic
:::

## 带图标的

:::demo 默认展示传入文本的第一个字作为图标
cascader/icon
:::


## 可清空

:::demo 通过设置```clearable```设置输入框清空
cascader/clearable
:::
